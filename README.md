# Taha-mozafari-web
# 🌐 طاها مظفری | سایت شخصی (Personal Website)

این پروژه دوم من و اولین سایت شخصی حرفه‌ای‌ام است. در این سایت، مهارت‌ها، پروژه‌ها، گواهی‌نامه‌ها و راه‌های ارتباطی خود را به نمایش گذاشته‌ام.

## 🚀 ویژگی‌های پروژه

- **طراحی کاملاً ریسپانسیو**: با استفاده از `flexbox`, `grid` و `media queries`
- **تم دارک/لایت**: قابلیت تغییر تم به همراه ذخیره‌سازی انتخاب کاربر در مرورگر (localStorage)
- **انیمیشن‌های نرم**: با `transition` و `transform`
- **سئو اولیه**: استفاده از متا تگ‌ها، `sitemap.xml` و `robots.txt`
- **گالری گواهی‌نامه‌ها**: نمایش مدارک با قابلیت تست اصالت و دانلود
- **فرم تماس و لینک‌های ارتباطی**
